# 游玩 Cytoid

作为 Cytoid 玩家的你，可以：

- 在 __社区__ 中浏览并下载由玩家们创作的超过 4000 个关卡
- 与朋友和其他玩家在关卡的 __排行榜__ 上竞争
- 在实力增长的同时提升自己的 __Rating__ 技术值，在玩家榜上一争高下
- 在不定期开展的季节与合作 __活动__ 中赢取角色与关卡奖励
- 在循序渐进的 __段位__ 模式中获得认定，证明自己的实力
